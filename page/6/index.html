为什么把析构函数定义为虚函数？

发表于 2016-03-23   |   分类于 C++

先来看一个例子：

会输出什么呢？再改一下。

答案是第一段程序输出

~derived
~base
第二段程序则输出为
~base

执行析构函数时先调用派生类的析构函数，然后才调用基类的析构函数。如果析构不是虚函数，而程序执行时又要通过基类的指针去销毁派生类的动态对象，那么用delete销毁对象时，
只调用了基类的析构，未调用派生类的析构，这样会造成销毁对象不完全。
如果不需要基类对派生类及对象进行操作的时候，则不必定义虚函数，因为定义了虚函数后编译器会给类添加一个虚函数表存放虚函数指针，增加了存储空间。

常见C++面试题

发表于 2016-03-23   |   分类于 C++

1.C++多态性有哪些？
静态：编译时刻多态，通过重载函数，类模板多态实现
动态：运行时刻多态，通过虚函数类继承实现
2.C++中类型转换有哪些？
static_cast :最常用的类型转换，如把int转换为float
const_cast :用于取出const属性，例如把const指针转换为非const指针
dynamic_cast :只在转换一个具有至少一个虚函数的类时合法
reinterpret_cast :重新解释数据的二进制，很少用
3.什么是浅拷贝深拷贝？
浅拷贝：用一个现成的对象创建另一个对象时，只拷贝了成员，没有拷贝分配给现成对象中成员的资源
深拷贝：定义对象的拷贝函数，使之资源也拷贝
4.const int i怎么做到的只可读？
编译器相关，优化可能让其直接转为一个常量代入
5.OFFSETOF(s,m)的宏定义，s是结构类型，m是S的成员，求m在s中的偏移量

#define OFFSETOF(s,m) ((size_t)(&s->m)-(size_t)(&s))
#define OFFSETOF(s,m) (size_t)&(((s*)0)->m)
取地址符号 & 决定表达式不会出现访问内存错误，因为将0转化为结构体指针类型后是不能访问的地址。->的本质就是在0后面加上结构体m的偏移量。

2016网易实习生招聘笔试编程第一题
动态规划掌握不够好，挂了这题。

Michael喜欢滑雪百这并不奇怪，因为滑雪的确很刺激。可是为了获得速度，滑的区域必须向下倾斜，而且当你滑到坡底，
你不得不再次走上坡或者等待升降机来载你。Michael想知道载一个区域中最长底滑坡。
区域由一个二维数组给出。数组的每个数字代表点的高度。下面是一个例子

一个人可以从某个点滑向上下左右相邻四个点之一，当且仅当高度减小。在上面的例子中，一条可滑行的滑坡为24-17-16-1。
当然25-24-23-…-3-2-1更长。事实上，这是最长的一条。

思路：题目意思其实很简单，给出一个二维数组，在其中找出一个点，是它能找到一条高度依次下降的路径并使得这条路径最长。
利用动态规划的变形
求解从每个点出发的最长路径长度，
求解从每一个出发点最长路径过程就是一个不断求最优解的过程。
关键就是理解动态规划和递归的关系！
摘自算法导论:

通常按如下4个步骤设计一个动态规划算法
1.刻画一个最优解的结构特征
2.递归地定义最优解的值 –重点理解
3.计算最优解的值，通常采用自底向上的方法
4.利用计算出的信息构造一个最优解

在这题中，一个点的最大高度等于周围点即上下左右点最大高度加1。需要注意各种判断条件。
