常见C++面试题

发表于 2016-03-23   |   分类于 C++

1.C++多态性有哪些？
静态：编译时刻多态，通过重载函数，类模板多态实现
动态：运行时刻多态，通过虚函数类继承实现
2.C++中类型转换有哪些？
static_cast :最常用的类型转换，如把int转换为float
const_cast :用于取出const属性，例如把const指针转换为非const指针
dynamic_cast :只在转换一个具有至少一个虚函数的类时合法
reinterpret_cast :重新解释数据的二进制，很少用
3.什么是浅拷贝深拷贝？
浅拷贝：用一个现成的对象创建另一个对象时，只拷贝了成员，没有拷贝分配给现成对象中成员的资源
深拷贝：定义对象的拷贝函数，使之资源也拷贝
4.const int i怎么做到的只可读？
编译器相关，优化可能让其直接转为一个常量代入
5.OFFSETOF(s,m)的宏定义，s是结构类型，m是S的成员，求m在s中的偏移量

#define OFFSETOF(s,m) ((size_t)(&s->m)-(size_t)(&s))
#define OFFSETOF(s,m) (size_t)&(((s*)0)->m)
取地址符号 & 决定表达式不会出现访问内存错误，因为将0转化为结构体指针类型后是不能访问的地址。->的本质就是在0后面加上结构体m的偏移量。
