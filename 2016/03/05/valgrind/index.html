Valgrind 安装与使用

发表于 2016-03-05   |   分类于 Linux

valgrind是Linux平台下著名的内存泄漏检测工具

安装流程：

  1. 到www.valgrind.org下载最新版valgrind-..*.tar.bz2

  2. 解压安装包：tar –jxvf valgrind-3.2.3.tar.bz2

  3. 解压后生成目录valgrind-3.2.3

  4. cd valgrind-3.2.3

  5. ./autogen.sh
    如果出现：./autogen.sh: 1: eval: aclocal: not found错误，则apt-get install autoconf

  6. ./configure;配置Valgrind，生成MakeFile文件，具体参数信息详见INSTALL文件。一般只需要设置–prefix=/where/you/want/it/installed

  7. Make；编译Valgrind

  8. make install；安装Valgrind
