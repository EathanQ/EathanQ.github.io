win7(x64)安装scrapy框架

Python

安装:
1.python 2.7 安装版本： python-2.7.5.amd64.msi
http://www.python.org/downloads/windows/
add C:\Python27 and C:\Python27\scripts in your PATH.
检验是否安装成功方法为：win+R运行CMD输入python
2.OpenSSL
http://slproweb.com/products/Win32OpenSSL.html
add C:\OpenSSL-Win64\bin in your PATH
3.SetupTools
win7 64位必须使用ez_setup.py进行安装。
方法是下载ez_setup.py后，在cmd下执行 python ez_setup.py，即可自动安装setuptools。
win7 32位可以下载setuptools-0.6c11.win32-py2.7.exe
4.pywin32
https://sourceforge.net/projects/pywin32/files/pywin32/
选择最新的下载
5.pip:Python的包管理工具
可以参考：http://blog.csdn.net/liuchunming033/article/details/39578019
下载pip的安装包get-pip.py，下载地址：https://pip.pypa.io/en/latest/installing.html#id7
python get-pip.py
6.twisted
https://pypi.python.org/pypi/Twisted
Twisted-15.5.0-cp27-none-win_amd64.whl (md5)
运行pip install Twisted-15.5.0-cp27-none-win_amd64.whl 安装
7.zope interface
https://pypi.python.org/pypi/zope.interface/4.1.0
选择与安装的python对应版本安装
8.lxml
lxml-3.4.4-cp27-none-win_amd64.whl
9.Scrapy
cmd中输入pip install scrapy

错误处理：
1.unable to find vcvarsall.bat
安装VCForPython27.msi
2.其他安装错误
检查安装版本是否与安装python版本对应，例如如果安装python-win32-2.7，那么应该安装32位对应2.7python
